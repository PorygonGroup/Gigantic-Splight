# Gigantic-Splight

---
This project aims to simulate 3D fluids based on Taichi framework.

## Getting Started

- `w/a/s/d`: move camera horizontally
- `space/shift`: move camera vertically
- `arrow keys`: move camera viewpoint
- `r`: reset camera
- `t/y`: enable/disable board movement
- `b`: ball pov
